# Hasan Misaii — CV Website

Static CV/portfolio website ready for GitHub Pages.

## Files

- `index.html` — main website content
- `style.css` — visual theme and responsive layout
- `script.js` — theme toggle, mobile menu, publication filtering, contact form mailto fallback
- `assets/avatar.svg` — placeholder avatar used by the hero section
- `.nojekyll` — tells GitHub Pages to serve the static files directly

## Deploy with GitHub Pages

1. Create a GitHub repository named `hasanmisaii.github.io` for a personal GitHub Pages site, or any repository name for a project site.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. Your site will be available at `https://hasanmisaii.github.io/` if using the personal-site repository name.

## Before publishing

- Replace the placeholder Google Scholar URL in `index.html` with your real Google Scholar profile.
- Consider replacing the placeholder avatar with a professional photo.
- Consider whether to publish referee emails publicly, or replace them with “Available upon request.”
